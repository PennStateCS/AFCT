API Documentation

Endpoint: POST /api/public/login

Content-Type: application/json

Request Body:

{
  "email": "user@example.com",
  "password": "password123"
}

Success Response (200):

{
  "user": {
    "id": "user_id",
    "email": "user@example.com",
    "firstName": "First",
    "lastName": "Last",
    "role": "ADMIN"
  }
}

Error Response (401):

{
  "error": "Invalid credentials"
}
